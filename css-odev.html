CSS Temel Ödevi

Verilen benzer bir düzenle basit bir web sayfası oluşturun. CSS'in temel özelliklerini (display, width, height, margin, padding, border, selector) uygulayın. Sadece HTML5 + CSS3 kullanın, hazır framework kullanmak yasak (Bootstrap, Tailwind vb.).

İçerik

Bu alanda örnek içerik metinleri bulunmaktadır. CSS özellikleriyle bu metinler arasında boşluklar (line-height) ve hizalama (text-align) düzenlenmiştir.

Önce CSS ile kutunun / alanın yan yana gelmesini, ardından da alta yerleşmesini öğreneceksiniz. Şimdi ise CSS seçicilerle kullanarak grup ve hiyerarşik seçimler yapmayı göreceğiz.

Yan Menü / Bilgi

Bu alan yan menü veya bilgi kısmıdır. CSS ile genişlik (width), yükseklik (height), kenarlık (border), iç/dış(margin/padding) ve kutuları düzenleyebiliriz.

Öğrendiklerimizi hiyerarşik ve grup seçicilerle uygulayıp farklı bölümlere etki ettirebiliriz.
